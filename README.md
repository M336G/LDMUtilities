# LDM Utilities
A <cg>utility mod</c> for <cg>Geometry Dash</c>'s <cg>Low Detail Mode</c> on levels.

## Features
- **<cg>Auto LDM</c>** (<cr>off</c> by default): automatically enables Low Detail Mode on levels that support it (does not override your existing toggle preference per level).
- **<cg>Disable LDM button</c>** (<cg>on</c> by default): disables the Low Detail Mode button on levels that do not support Low Detail Mode.

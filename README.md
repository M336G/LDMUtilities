# LDM Utilities
A <cb>utility mod</c> for <cb>Geometry Dash</c>'s <cb>Low Detail Mode</c> on levels.

## Features
- **<cl>Auto LDM</c>** (<cr>off</c> by default): automatically enables Low Detail Mode on levels that support it (does not override your existing toggle preference per level).
- **<cl>Disable LDM button</c>** (<cg>on</c> by default): disables the Low Detail Mode button on levels that do not support Low Detail Mode.
